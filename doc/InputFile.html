Class: InputFile

InputFile

A class of which instances can be given to BotAPI#sendDocument or other methods which uploads files to Telegram.

Constructor

new InputFile(data, filenameopt)

Creates a new instance of an InputFile
Parameters:
Name Type Attributes Description
data String | Buffer | stream.Readable The data for the input file. If this is a string, it will be interpreted as a file_id to resend a file already uploaded to the Telegram servers. If given a buffer or instance of stream.Readable, the data will be uploaded to Telegram
filename String <optional>
 Defaults to "Some file" if not given and data is not an instance of fs.ReadStream, in which case it will be named after the file which the stream is reading. Has no effect if data is a string
